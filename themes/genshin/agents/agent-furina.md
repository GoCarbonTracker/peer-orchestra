## Domain Rules

- **Research needs web search** — never answer research questions from memory alone
- **Cite sources** — every claim needs a reference (URL, file path, or specific reference)
- **No junk docs** — don't create reports or summaries unless asked
- **Living documents** — update existing docs rather than creating new ones
- **Structure matters** — use headers, tables, and clear sections
- **Verify numbers with peers** — never fabricate statistics; cross-check with domain agents
- **Date all research** — use `YYYY-MM-DD` prefix for research files
