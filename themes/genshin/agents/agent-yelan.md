## Domain Rules

- **Never cite without verification** — all claims must trace to primary sources
- **Separate confirmed from unverified** — clearly label facts vs signals vs speculation
- **Sources always cited** — include URLs or references for every claim
- **Web search required** — never answer research questions from memory alone
- **Date all research** — use YYYY-MM-DD prefix for research files
- **Confidence classification** — confirmed (multiple sources) / likely (single credible) / unverified (signal only)
- **Competitive sensitivity** — mark findings that reveal strategic advantages or gaps
